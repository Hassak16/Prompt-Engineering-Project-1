Overview:

This project demonstrates how to use Prompt Engineering techniques to extract 
structured data from unstructured text using Large Language Models (LLMs).
The main goal is to convert messy text into clean JSON output using well-designed prompts.

Features:

• Zero-Shot Prompting
• Few-Shot Prompting
• Data Extraction from Raw Text
• JSON Output Formatting
• Delimiter-Based Prompt Design
• Deterministic Response Generation

Project Objective:

Create prompts that:
1. Accept unstructured text as input.
2. Extract required information accurately.
3. Return only valid JSON output.
4. Avoid unnecessary explanations or conversational text.

Technologies used in this project are:

• Prompt Engineering
• Large Language Models (LLMs)
• JSON
• GitHub

Example Input:

My name is Sara Khan.
Email: sara@gmail.com
Phone: 03001234567

Example Output:

{
  "name": "Sara Khan",
  "email": "sara@example.com",
  "phone": "03001234567"
}

Learning Outcomes:

• Understanding Zero-Shot Prompting
• Understanding Few-Shot Prompting
• Designing effective prompts
• Working with structured JSON outputs
• Improving AI response reliability
